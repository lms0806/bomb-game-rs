# bomb-game-rs

크레이지아케이드 스타일의 간단한 폭탄 게임입니다.  
[Rust `windows` 크레이트](https://docs.rs/windows/latest/windows/)와 Win32 API(GDI)로 만든 Windows 데스크톱 게임입니다.

## 요구 사항

- Windows 10 이상
- [Rust](https://www.rust-lang.org/) **1.96.0** (Cargo 포함)

프로젝트 루트의 `rust-toolchain.toml`이 Rust 1.96.0을 사용하도록 설정되어 있습니다.

## 실행 방법

```powershell
git clone <저장소 URL>
cd bomb-game-rs
cargo run
```

릴리스 빌드:

```powershell
cargo run --release
```

## 조작법

| 키 | 동작 |
|---|---|
| 방향키 (↑↓←→) | 플레이어 이동 |
| 스페이스바 | 폭탄 설치 |
| 스페이스바 (게임 오버 시) | 게임 재시작 |

## 게임 규칙

- 맵 위를 이동하며 **빨간 몬스터**를 폭탄으로 처치합니다.
- 폭탄은 설치 후 약 **1.5초** 뒤 **십자 방향**으로 폭발합니다.
- 기본적으로 **동시에 폭탄 1개**만 설치할 수 있습니다.
- 몬스터는 랜덤 위치에 스폰되며, 최대 **8마리**까지 맵에 존재할 수 있습니다.
- **자신이 설치한 폭탄의 폭발 범위** 안에 들어가면 게임 오버입니다.

## 아이템

몬스터를 처치하면 **10% 확률**로 아이템이 드랍됩니다.  
아이템은 몬스터가 있던 위치에 떨어지며, 폭발 연출이 끝난 뒤 나타납니다.

| 색상 | 효과 | 최대 |
|---|---|---|
| 초록 | 폭발 범위 +1 | 5칸 |
| 노랑 | 동시 설치 가능한 폭탄 +1 | 5개 |

아이템 칸으로 이동하면 자동으로 획득됩니다.

## 게임 오버 & 재시작

폭발 범위에 플레이어가 포함되면 게임 오버됩니다.  
화면에 **「스페이스바를 눌러 재시작」** 안내가 표시되며, 스페이스바를 누르면 처음 상태로 다시 시작합니다.

## 화면 구성

| 요소 | 색상 |
|---|---|
| 배경 | 파랑 |
| 플레이어 | 흰색 |
| 폭탄 | 검정 |
| 몬스터 | 빨강 |
| 폭발 | 주황 |
| 범위 증가 아이템 | 초록 |
| 폭탄 개수 증가 아이템 | 노랑 |

## 기술 스택

- **언어**: Rust (2021 edition)
- **플랫폼**: Win32 (`windows` 0.62)
- **렌더링**: GDI (`FillRect`, `DrawTextW` 등)
- **게임 루프**: Win32 메시지 루프 + 타이머 (`WM_TIMER`)
